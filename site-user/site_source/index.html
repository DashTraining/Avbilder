Azure on the cheap?

Free is possible but scalability, redundancy, identity management is where upgrades begin.

This demo works on the Azure Static Web Apps Free tier and uses built-in Microsoft authentication, managed Azure Functions. Services like Microsoft Entra External ID, custom Azure Functions, scalable SWA Standard could be seen as the clear upgrade path.
